# CogniKids - Play That Builds Big Minds

A beautiful, responsive single-page landing page for CogniKids, an educational app designed for toddlers. This project showcases a modern, child-friendly interface with interactive game demonstrations, educational content, and comprehensive information about the app's features.

## üéØ Project Overview

CogniKids is a landing page designed to promote an educational mobile application for young children. The page features an interactive app simulator that demonstrates ABC learning, number recognition, and shape identification games. Built with a focus on user experience and visual appeal, the landing page effectively communicates the app's value proposition to parents and caregivers.

The project includes:
- Interactive app simulator with playable mini-games
- Responsive design that works on all devices
- Modern UI with smooth animations and transitions
- Educational content sections highlighting curriculum alignment
- Parental control information
- Pricing plans and testimonials

## üöÄ Setup Instructions

### Prerequisites

No special prerequisites required! This is a static website that runs entirely in the browser.

### Installation

1. **Clone or Download** the project files to your local machine

2. **Open the project**
   - Simply open `index.html` in any modern web browser
   - Or use a local development server for better experience

3. **Using a Local Development Server (Optional)**
   
   **Option A: Using Python**
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Python 2
   python -m SimpleHTTPServer 8000
   ```
   Then open `http://localhost:8000` in your browser.

   **Option B: Using Node.js (http-server)**
   ```bash
   npx http-server
   ```
   Then open the URL shown in the terminal (usually `http://localhost:8080`).

   **Option C: Using VS Code Live Server**
   - Install the "Live Server" extension in VS Code
   - Right-click on `index.html` and select "Open with Live Server"

### Running the Project

1. Open `index.html` in your web browser
2. Navigate through the page using the top navigation menu
3. Interact with the app simulator in the hero section
4. Test the responsive design by resizing your browser window

- Firefox (latest)

- Mobile browsers (iOS Safari, Chrome Mobile)
